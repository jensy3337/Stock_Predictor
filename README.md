# IndiaMarket Pro

Dark-themed React app for Indian market analysis and prediction.

## Run locally

```bash
npm install
npm run dev
```

## Features
- Dashboard + Education pages
- NIFTY / SENSEX / NIFTY BANK + MCX instruments
- 5-month candlestick chart and 2-month linear-regression prediction overlay
- Sector filters, search, watchlist, top gainers/losers, heatmap, market sentiment
- News feed, price alerts, localStorage persistence, 15-second auto-refresh
